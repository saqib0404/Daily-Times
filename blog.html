Blog-1: What are the differences between var, let, const ?

Answer: The main differences between var, let, const are:

var is scoped to the immediate function body. Whereas let and const are scoped to the immediate enclosing block denoted by { }.
var can be redeclared. Whereas let and const can not be redeclared.
var and let can be reassigned but const can not.

Blog-2: What are the differences between Arrow Function and Regular Function ?

Answer: The main differences between Arrow function and Regular function is its syntax.

Inside of a regular JavaScript function, 'this' value is dynamic. No matter how or where being executed, 'this' value inside of an arrow function always equals this value from the outer function. In other words, the arrow function resolves this lexically.
In the regular function we have to right 'return' keyword in order to return anything. In case of arrow function we can do so but if the code is of one line we can simply avoid 'return' keyword as it is going to return anyway,

Blog-3: What are the differences between Map, ForEach, Filter and Find ?

Answer: Map, ForEach, Filter and Find are all ES6 properties.

map() and filter() returns a new array. Whereas forEach() returns undefined and find() return a single value.
map() performs it's activity then returns. forEach() also performs it's activity but doesn't return. filter() checks the condition then returns the matched results in an array. find() checks the condition and returns only the first matched element.

Blog-4: Why do we use Template String ?

Answer: Template String is a feature of ES6.

Template String is used most commonly for string interpolation. Template literals are enclosed by backtick (`) characters instead of double or single quotes. Along with having normal strings, template literals can also contain other parts called placeholders, which are embedded expressions delimited by a dollar sign and curly braces: ${expression}. The strings and placeholders get passed to a function â€” either a default function, or a function you supply. The default function (when you don't supply your own) just performs string interpolation to do substitution of the placeholders and then concatenate the parts into a single string. For this we use template string.
